📦 DeliverTrack

Real-time delivery management for your team. Follow the steps below to connect your free Firebase database — takes about 5 minutes.

⚠️ Cannot load Firebase scripts. This usually means no internet connection, or your browser is blocking external scripts.

Fix: Open this file in Chrome or Safari on your phone, make sure you have internet, then try again. Or host it on Netlify for best results.
Step 1

Go to console.firebase.google.com → Create a project (any name, disable Google Analytics if asked).

Step 2

Inside your project → click Build → Realtime Database → Create Database → choose any location → select Start in test mode → Enable.

Step 3

Go to Project Settings (⚙ gear icon) → scroll to Your apps → click </> Web → register your app → copy the firebaseConfig object and paste it below.
